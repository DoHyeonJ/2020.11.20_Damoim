# 1. 프로젝트 개요
- 동호회 모임 서비스 
- 본인의 활동지역과 관심 키워드를 등록하여 그에 맞는 동호회를 검색하고 가입하고 모임을 가질 날짜와 시간을 정하여 모임을 할 수 있는 서비스

<br>



# 2. 사용 기술 및 기능
- Java 11 
- Spring Boot 2.3.4 
- postgresql
- Jqyery, Ajax
- HTML, CSS, JS
- Thymeleaf
